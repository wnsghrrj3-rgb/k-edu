🤔 한 명이야, 여러 명이야?
⭕ 👩 This is my mom.한 명(엄마) → This is ✨
⭕ 👨👩 These are my parents.두 명(부모님) → These are ✨
❌ 👨👩 This is my parents.두 명인데 This is? → 틀림
❌ 👩 These are my mom.한 명인데 These are? → 틀림
🚨 "한 명 → This is / 여러 명 → These are" 규칙!
✅ 한 명 👩: This is my mom.
✅ 여러 명 👨👩: These are my parents.
💡 This (이것/이 분) = 하나 / These (이것들/이 분들) = 여럿!
