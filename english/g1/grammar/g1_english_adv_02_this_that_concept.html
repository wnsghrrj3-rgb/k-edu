a
자음 앞
(A, E, I, O, U 빼고)
a cat 🐱
a dog 🐶
a book 📖
an
모음 앞
(A, E, I, O, U)

an elephant 🐘
an orange 🍊
🐶
A, E, I, O, U 이 다섯 글자는 모음!
이 앞에서는 발음하기 편하라고
an으로 바꿔줘!
a 쓰는 예시
🔵 a + 자음 시작
🐱
a cat 🔊고양이 한 마리
🐶
a dog 🔊개 한 마리
📖
a book 🔊책 한 권
✏️
a pencil 🔊연필 한 자루
🐶
c, d, b, p... 전부 자음!
그래서 전부 a를 붙였어!
an 쓰는 예시
🟣 an + 모음 시작 (A/E/I/O/U)
🍎
an apple 🔊사과 한 개 (A로 시작)
🐘
an elephant 🔊코끼리 한 마리 (E로 시작)
🍊
an orange 🔊오렌지 한 개 (O로 시작)
🥚
an egg 🔊달걀 한 개 (E로 시작)
🐶
"a apple"이라고 하면 입이 불편해! 😣
그래서 an apple로 발음을 부드럽게!
⚠️ 자주 틀려!
🚨 헷갈리는 지점
❌ 틀림 This is a apple.
✅ 맞음 This is an apple.
❌ 틀림 That is an cat.
✅ 맞음 That is a cat.
❌ 틀림 This is a Kay.
✅ 맞음 This is Kay.
🐶
사람 이름 앞에는 a/an 안 붙여!
케이는 한 명뿐이니까 번호표가 필요 없어!
