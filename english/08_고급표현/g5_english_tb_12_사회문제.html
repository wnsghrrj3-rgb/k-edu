🌍 개념 학습 — 문제-해결 구조
We should ~ / If we don't ~, then ~
사회 문제를 말하고 해결책을 제안하는 패턴
🔊 We should reduce plastic waste to protect the ocean.
🔊 If we don't recycle, then our planet will be in danger.
💡 사회 문제 어휘: pollution(오염), waste(쓰레기), protect(보호), reduce(줄이다)
🌍 문제-해결 글쓰기 구조
Problem → Cause → Solution → Expected Result
❗ Problem: There is too much plastic in the ocean.
🔍 Cause: People throw away plastic bottles every day.
💡 Solution: We should use reusable bottles.
✅ Result: If we do this, the ocean will be cleaner.
