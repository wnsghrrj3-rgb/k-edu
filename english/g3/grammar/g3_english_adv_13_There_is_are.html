There is / There are
"~이 있다" 존재 표현 + some / any / no
📏 There is vs There are
There is
단수 / 셀 수 없는 명사
There is a cat 🐱 on the sofa.
There is some water 💧.
There are
복수
There are two dogs 🐕🐕.
There are many books 📚.
🔄 부정·의문 + some / any / no
❌ 부정: There is not (isn't) / are not (aren't)
There aren't any apples. = There are no apples.
❓ 의문: Is there~? / Are there~?
Is there a park near here? — Yes, there is.
💡 긍정: some / 부정·의문: any
