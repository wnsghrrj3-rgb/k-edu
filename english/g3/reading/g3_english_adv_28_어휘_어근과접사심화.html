📖 접두사 (Prefix) 복습 + 확장
접두사 = 단어 앞에 붙는 조각 🧩
뜻을 바꾸거나 추가해요!
접두사뜻예시
un-~아닌 (not)un + happy = unhappy
re-다시 (again)re + write = rewrite
dis-~아닌 (not)dis + agree = disagree 🆕
mis-잘못 (wrongly)mis + understand = misunderstand 🆕
over-지나치게 (too much)over + sleep = oversleep 🆕
📖 접미사 (Suffix) 복습 + 확장
접미사 = 단어 뒤에 붙는 조각 🧩
품사를 바꿔요!
접미사만드는 품사예시
-ly부사quick → quickly
-ful형용사 (가득한)hope → hopeful
-less형용사 (없는)hope → hopeless
-ment명사move → movement 🆕
-tion명사act → action 🆕
-ness명사kind → kindness
🔧 단어 조립 공장!
dis- + agree = disagree
동의하지 않다 (not agree)
educate + -tion = education
교육 (명사)
💡 접두사는 뜻을 바꾸고, 접미사는 품사를 바꿔요!
