There is / There are

존재 표현 + some · any · no

4학년 Track B-1 · 13차시

📖 개념 1 — There is / There are

There is

단수 / 셀 수 없는 명사
"There is a cat."
"There is water."

There are

복수 명사
"There are three cats."
"There are many books."

💡 "~이 있다"를 말할 때 사용! 부정: There isn't / aren't
📖 개념 2 — some · any · no
✅ 긍정문: There are some apples. (몇 개의)
❓ 의문문: Are there any apples? (있나요?)
🚫 부정문: There aren't any apples.
🚫 부정문: There are no apples. (= aren't any)
🔑 some = 긍정문 / any = 의문·부정문 / no = 부정 (동사는 긍정형!)
📖 개념 3 — 의문문 만들기
Is / Are there ~?
is/are를 앞으로 보내면 의문문!
🏞️ Is there a park near here?
🏫 Are there any students in the room?
💬 응답: Yes, there is. / No, there aren't.
