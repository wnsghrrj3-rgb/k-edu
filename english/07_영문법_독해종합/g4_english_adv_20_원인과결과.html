독해 전략 4 — 원인과 결과

because · so · therefore 관계 파악

4학년 Track B-2 · 20차시

📖 개념 — 원인(Cause)과 결과(Effect)

💡 Cause (원인)

왜 일어났는가?
because 뒤에!

➡️ Effect (결과)

무슨 일이 일어났는가?
so / therefore 뒤에!

It rained heavily, so the game was canceled.
The game was canceled because it rained heavily.
💡 같은 내용이지만 방향이 반대! because = 원인 쪽 / so = 결과 쪽
