📌 어원 학습의 3가지 힘

🔹 하나의 어근으로 10~20개 단어를 한 번에
🔹 처음 보는 단어도 의미 추론 가능
🔹 수능·토플·특목고 어휘의 70%가 라틴·그리스 어근
🔤 핵심 어근 그룹 1
-ject- = throw (던지다)
→ reject (re+ject = 되던지다 → 거부하다)
→ project (pro+ject = 앞으로 던지다 → 계획, 투영하다)
→ inject (in+ject = 안으로 던지다 → 주입하다)

-spect- = look (보다)
→ inspect (in+spect = 안을 보다 → 검사하다)
→ respect (re+spect = 다시 보다 → 존경하다)
→ perspective (per+spect+ive = 끝까지 보는 것 → 관점)

-duc/duct- = lead (이끌다)
→ conduct (con+duct = 함께 이끌다 → 수행하다, 지휘하다)
→ reduce (re+duce = 되이끌다 → 줄이다)
→ introduce (intro+duce = 안으로 이끌다 → 소개하다)

🔤 핵심 어근 그룹 2
-cred- = believe (믿다)
→ incredible (in+cred+ible = 믿을 수 없는)
→ credible (cred+ible = 믿을 만한)
→ credit (cred+it = 신뢰, 학점)

-port- = carry (나르다)
→ transport (trans+port = 건너 나르다 → 운송하다)
→ export (ex+port = 밖으로 나르다 → 수출하다)
→ import (im+port = 안으로 나르다 → 수입하다)

-vis/vid- = see (보다)
→ visible (vis+ible = 볼 수 있는)
→ evidence (e+vid+ence = 밖으로 보이는 것 → 증거)
→ revise (re+vis+e = 다시 보다 → 수정하다)
